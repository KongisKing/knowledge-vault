## Definition
A compounding competitive advantage where more usage generates more data, which improves the product, which drives more usage. Each cycle reinforces the next.

## In Agentic AI
Agent memory is the flywheel mechanism:
- Every user interaction teaches the agent more about that user
- Preferences, patterns, and habits accumulate over time
- The agent becomes increasingly personalised and hard to replace
- Nobody else has access to this dataset

## Why It Matters Strategically
> Without memory → your agent is replicable by anyone with the same tools and model
> With memory → you have a proprietary, compounding dataset nobody else can replicate

This is why model providers are racing to own the memory layer — they want the flywheel to live in their platform, creating lock-in stronger than any API moat.
